    Use MkDocs to Build Your First Personal Website¶

    MkDocs is a fast, simple and downright gorgeous static site generator that's geared towards building project documentation. Documentation source files are written in Markdown, and configured with a single YAML configuration file.

    Official Website: https://www.mkdocs.org/

    2 Install MkDocs¶

    1.Enter your directory.

    1
    cd github-pages

    2.Create a virtualized python environment, here we use

    Tip

    Using material theme may cause errors without this environment.

    pipenv install

    3.Install MkDocs.

    1
    pipenv install mkdocs

    4.Enter the python virtual environment

    1
    pipenv shell

    5.Check if the installation is succeessful

    1
    mkdocs --version

    3 Create MkDocs Project¶

    Create a new project in the current directory (here's ).

    1
    mkdocs new .

    Then compile the document

    1
    mkdocs build

    4 Preview and Deploy¶

    Preview

    1
    mkdocs serve [-a 0.0.0.0:8000]

    The -a option is used to specify the binding , if not specified, the default is localhost:8000. If you find a problem with the preview, you can modify the document source file directly, and MkDocs will automatically update the preview when it detects the file change.

    Deploy

    1
    mkdocs  gh-deploy --clean
